# ats-cv üìÑ

A smart web application that analyzes resumes and checks their compatibility with Applicant Tracking Systems (ATS). It provides feedback and improvement suggestions to help users optimize their CVs for better job application results.

## üöÄ Introduction

**ats-cv** is an intelligent, user-friendly web application designed to empower job seekers by ensuring their resumes are ATS-compatible. Applicant Tracking Systems are widely used by employers to screen resumes, and non-optimized CVs can get rejected before human eyes ever see them. With ats-cv, users receive actionable insights, compatibility scores, and suggestions to make their resumes stand out and successfully pass ATS filters.

---

## ‚ú® Features

- **Resume Analysis**: Upload your CV and get a detailed compatibility report.
- **ATS Compliance Checks**: Automatic detection of formatting issues and missing keywords.
- **AI-Powered Suggestions**: Personalized feedback and improvement tips.
- **Career Insights Dashboard**: Track your optimization progress over time.
- **User Settings**: Configure your profile and preferences for tailored recommendations.
- **Secure & Private**: All data is processed securely and privately.

## ‚ö° Installation

1. **Clone the repository**  
   ```bash
   git clone https://github.com/Imranch4/ats-cv.git
   cd ats-cv
   ```

2. **Create and activate a virtual environment**  
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

3. **Install dependencies**  
   ```bash
   pip install -r requirements.txt
   ```

4. **Set environment variables**  
   - Create a `.env` file in the project root.
   - Add your OpenAI API key and other config:
     ```
     OPENAI_API_KEY=your_api_key_here
     OPENAI_BASE_URL=https://api.openai.com/v1
     ```

5. **Run the application**  
   ```bash
   streamlit run app.py
   ```

## üìñ Usage

1. **Launch the app**
